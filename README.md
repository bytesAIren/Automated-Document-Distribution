# Automated Document Distribution (Excel + Google Apps Script)

This project automates the creation and delivery of personalized PDF documents.
It combines an Excel/VBA pipeline for document generation with a Google Apps Script
automation for Gmail to send customized emails securely.

## Features
- Generates personalized PDF letters from Excel data
- Supports multiple Word templates (V1, V2, V3)
- Automatically sends Gmail messages with individual subject and body
- Safe sandbox mode (manual confirmation before sending)
- No external dependencies, all native Office + Google tools

## Why this project
Born from a real workflow optimization need in a commercial back-office context,
this project demonstrates how to connect desktop automation with cloud-based tools
without complex infrastructure.
